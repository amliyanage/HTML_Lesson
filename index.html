Hypertext Markup Language 5

HTML5 is the latest version of HyperText Markup Language

Hypertext Markup Language revision 5 (HTML5) is markup language for the structure and presentation of World Wide Web contents.

HTML5 is a markup language used for structuring and presenting hypertext documents on the World Wide Web. It was the fifth and final major HTML version that is now a retired World Wide Web Consortium (W3C) recommendation. The current specification is known as the HTML Living Standard.
